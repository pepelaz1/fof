Eddystone This   boom   can   come   in   many   different   lengths   and   gives   more support for the weight as it is spread over more of the main line.
French These   are   the   easiest   boom   to   fix   to   the   main   line   as   all   you   have to   do   is   wrap   the   line   around.   The   boom   is   static   and   can   only   be moved manually. These   are   only   light   weight   and   not   suitable   when   fishing   for heavy   fish   as   they   tend   to   get   pulled   down   during   retrieving   and can weaken the main line.
Clements (Flying Collar) This   boom   needs   threading   onto   the   main   line   when   making   up the   end   tackle.   It   can   slide   up   and   down   the   main   line   or   made static by use of stoppers. Best used for heavy species.
Clements This   boom   has   to   be   threaded   onto   the   main   line,   it   does   not have the strength as the boom is not re-enforced.
Ashpole This   is   one   of   the   most   popular   of   boom   available;   strong   and inexpensive.
Booms A   boom   keeps   the   trace   line   away   from   the   main   line   and   this helps   stopping   tangles   occurring   especially   when   drifting   or   when using long traces. Used   with   a   Paternoster   rig   (the   weight   below   the   hook)   a   boom can   be   used   to   enable   positioning   the   bait   a   certain   distance above the sea bed. Used   as   part   of   a   Ledger   rig   (the   weight   above   the   hook)   they   can be   run   freely   to   enable   the   main   line   to   be   extended   to   any length.
